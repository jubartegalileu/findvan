## 🚀 Getting Started

### Quick Start (30 seconds)

1. **Open Claude Code**
2. **Type:** `/brad-frost` (or any agent name)
3. **Press:** Enter
4. **Agent activates** with full persona & commands
5. **Type:** `*help` to see all commands

### Alternative Activation Methods

```bash
/brad-frost                    # Slash command (fastest)
@brad-frost                    # Shorthand
/design:agents:brad-frost      # Full AIOS pattern
```

All three methods work identically.

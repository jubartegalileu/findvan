Activation wrapper for Brad Frost agent from squads/design.

```yaml
activation-instructions:
  - STEP 1: Load agent definition from: squads/design/agents/brad-frost.md
  - STEP 2: Execute full activation sequence from that file

  - STEP 4: Halt and await user input
```

---

To activate Brad Frost agent:

1. **Via shorthand:** `@brad-frost`
2. **Via slash command:** `/brad-frost`
3. **Via full pattern:** `/design:agents:brad-frost`

The agent definition in `squads/design/agents/brad-frost.md` will be loaded and fully activated.

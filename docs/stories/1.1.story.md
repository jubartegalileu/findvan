## 📋 Story Summary

Implement automated Google Maps scraper to capture school transportation business leads. This is the foundation for Module 1 and must be 100% complete before moving to Module 2 (Database).

**Goal:** Capture 100+ validated leads from Google Maps in < 5 minutes

---

## 📖 Story Description

### Problem/Need
We need an automated way to capture leads of school transportation businesses from Google Maps. Manual collection is too slow and error-prone.

### Solution
Build a Node.js scraper using Puppeteer that:
- Automates Google Maps searches
- Extracts lead data (name, phone, email, address, URL)
- Validates data quality
- Detects duplicates
- Saves to JSON for Module 2 import

### User Value
"As an SDR, I want to automatically capture 100+ leads from Google Maps so I can spend time on prospecting instead of manual data collection."

## 💡 Implementation Notes

### Why Puppeteer?
- Full browser automation
- Handles dynamic JavaScript-rendered content
- Easy error handling and retries
- Well-documented and stable

### Why JSON Before Database?
- Decoupled from Module 2
- Easy to validate independently
- Can review/clean data manually
- Backup of raw leads

### Data Schema
```json
{
  "id": "gm_20260227_001",
  "source": "google_maps",
  "name": "Van Escolar ABC",
  "phone": "11987654321",
  "email": "contato@vanabc.com",
  "address": "Av. Paulista, 1000, São Paulo",
  "city": "São Paulo",
  "company_name": "ABC Transportes",
  "url": "https://maps.google.com/...",
  "captured_at": "2026-02-27T08:30:00Z",
  "is_valid": true,
  "is_duplicate": false
}
```

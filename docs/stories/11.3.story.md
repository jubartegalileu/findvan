## Summary

Adicionar visibilidade operacional de campanhas nas telas WhatsApp/Campanhas com métricas de execução (enviadas, entregues, respostas, falhas) para acompanhamento diário do SDR.

## Acceptance Criteria

1. Visibilidade operacional
- [x] UI mostra resumo por campanha com enviados, entregues, respondidos e falhas.
- [x] Dados atualizam em cadência periódica sem bloquear navegação.
- [x] Estados vazios/sem dados apresentam fallback amigável.

2. Priorização e leitura rápida
- [x] Falhas e bloqueios aparecem com destaque de prioridade.
- [x] Operador distingue claramente campanha saudável vs campanha com risco.
- [x] Navegação entre WhatsApp/Campanhas mantém consistência visual.

3. Qualidade e regressão
- [x] Regressão visual nas rotas principais sem impacto crítico.
- [x] Sem erro de console/rede bloqueante no fluxo principal.

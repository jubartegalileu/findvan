## Summary

Implementar modo foco na tela de Leads para reduzir ruído e priorizar sinais críticos/altos da operação.

## Acceptance Criteria

1. Modo foco
- [x] Operador pode ativar/desativar modo foco com feedback visual claro.
- [x] Em modo foco, apenas sinais críticos/altos ficam em destaque.
- [x] Desativar modo foco restaura visualização completa sem perda de contexto.

2. Governança de prioridade
- [x] Fila/recomendações preservam ordenação por criticidade.
- [x] Itens não críticos permanecem acessíveis fora do modo foco.
- [x] Filtros e ações rápidas continuam funcionais.

3. Compatibilidade
- [x] Fluxos atuais de Leads permanecem íntegros.
- [x] Contratos de API permanecem compatíveis.
- [x] Gate arquitetural aprova modo foco.

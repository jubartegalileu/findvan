## API Contract (PO detail)

1. `GET /api/dashboard/weekly-performance`
- Response exemplo:
```json
{
  "status": "ok",
  "performance": {
    "messages_sent": 220,
    "delivery_rate": 94.1,
    "reply_rate": 18.6,
    "block_rate": 1.9,
    "series": [18, 26, 29, 31, 34, 42, 40]
  }
}
```

2. `GET /api/activity?limit=20`
- Response exemplo:
```json
{
  "status": "ok",
  "events": [
    {
      "id": "evt_1",
      "type": "scraper",
      "title": "Scraper Campinas/SP",
      "description": "8 leads inseridos",
      "created_at": "2026-02-28T12:00:00Z"
    }
  ]
}
```

## UI Behavior (PO detail)

- Widget de performance deve ficar visível sem scroll em 1280x720 no layout atual.
- Timeline deve ter ícone/cor por tipo e link `Ver histórico`.
- Ao erro de endpoint, mostrar estado de erro não bloqueante e manter resto do dashboard funcional.

# Story 7.2 — Scraper Coverage Map (RF-S5)

**Epic:** 7 (Wave 3 P2 Polish)
**Story ID:** 7.2
**Status:** Ready
**Priority:** P2 (Could Have)
**Complexity:** Média (5 pontos)
**RF:** RF-S5

## Summary

Adicionar visualização de cobertura no Scraper para mostrar estados/cidades coletados, média de leads por cidade e sugestão de próxima cidade.

## Acceptance Criteria

1. Widget de cobertura
- [ ] Exibir lista de estados com barra proporcional de cidades coletadas.
- [ ] Exibir total de cidades ativas.
- [ ] Exibir média de leads por cidade.

2. Sugestão de próxima cidade
- [ ] Exibir sugestão inicial (hardcoded) baseada em capitais ainda não cobertas.
- [ ] Sugestão deve ser legível e acionável no contexto do módulo Scraper.

3. Compatibilidade e Qualidade
- [ ] Widget não interfere no fluxo `Iniciar coleta` e `Agendamentos`.
- [ ] Sem criação de registros indevidos em `Execuções em andamento`.
- [ ] Contrato de cobertura documentado e validado.

## API Contract (PO detail)

1. `GET /api/scraper/coverage`
- Retorna payload com:
  - `states[]` (estado, cidades_coletadas, total_leads)
  - `active_cities_total`
  - `avg_leads_per_city`
  - `next_city_suggestion`

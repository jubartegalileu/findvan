# Story 7.3 — Leads Tags Customizáveis (RF-L8)

**Epic:** 7 (Wave 3 P2 Polish)
**Story ID:** 7.3
**Status:** Ready
**Priority:** P2 (Could Have)
**Complexity:** Média (5 pontos)
**RF:** RF-L8

## Summary

Implementar sistema de tags customizáveis em Leads para classificação manual, filtro e operação em lote.

## Acceptance Criteria

1. CRUD mínimo de tags no lead
- [ ] Adicionar tags no modal de detalhes (tab `Dados`).
- [ ] Remover tags existentes no lead.
- [ ] Autocomplete com tags já existentes.

2. Exibição e filtro
- [ ] Exibir tags no card do lead.
- [ ] Filtro por tag na tela de Leads.
- [ ] Filtro combinado com demais filtros (AND logic) permanece funcional.

3. Lote
- [ ] Ação em lote `Tagear` funcional para leads selecionados.
- [ ] Feedback de sucesso/erro após operação em lote.

4. Compatibilidade e Qualidade
- [ ] Fluxos P0/P1 de seleção em lote e status não regressam.
- [ ] Performance da lista continua estável em paginação atual.

## API Contract (PO detail)

1. `POST /api/leads/{id}/tags`
2. `DELETE /api/leads/{id}/tags/{tag}`
3. `POST /api/leads/batch/tag`
